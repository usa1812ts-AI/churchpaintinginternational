## 📱 PWA – Zum iPhone Homescreen hinzufügen

1. Link im **Safari** öffnen
2. Unten auf das **Teilen-Symbol** tippen (Quadrat mit Pfeil nach oben)
3. **„Zum Home-Bildschirm"** wählen
4. Namen bestätigen → **Hinzufügen**

Die App erscheint dann wie eine native App auf dem Homescreen.
